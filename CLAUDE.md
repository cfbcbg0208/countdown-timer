# CLAUDE.md — 카운트다운 타이머

## 무엇
특정 datetime을 입력하면 현재 시각부터 그 시각까지 **남은 시간**(미래) 또는
**경과 시간**(과거)을 1초마다 갱신해 보여주는 앱. 여러 목표를 저장·목록 표시.

## 확정 스택
- 바닐라 **웹/PWA** — HTML/CSS/JS(ES 모듈). **프레임워크·번들러·빌드 단계 없음.**
- 런타임 의존: **Node만**(테스트 `node --test`, dev서버 `node serve.mjs`). npm 패키지 0개.
- 데스크톱 우선. 추후 안드로이드는 같은 PWA를 '설치'/래핑으로 확장.

## 데이터 모델 (로컬 전용)
- 저장소: `localStorage` 키 `countdowns` = JSON 배열.
- 항목: `{ id, label, targetISO, createdAt }` (targetISO = ISO8601, 기기 로컬 타임존 기준).
- 외부 서버·동기화·계정 없음.

## 하드 제약
- 완전 오프라인 동작 · 개인정보는 로컬에만 · 외부 서비스 호출 금지.
- 의존성 최소(런타임 npm deps 0) · 빌드 산출물과 소스 분리.

## 작업 방식
- 계획 먼저 → 한 번에 작은 단위 1개 → 검증 → 작은 커밋.
- "됐다" 대신 **증거**(테스트 출력·실행 결과). 실패는 실패라고 말함.
- 순수 로직(src/time.js, store.js)은 `node:test` 자동 검증. DOM/렌더는 수동.

## 빌드 순서
M1 핵심로직(+테스트) → M2 단일표시 → M3 입력 둘다 → M4 여러개+영속 → M5 PWA → M6 다듬기.
